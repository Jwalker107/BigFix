Must return the format 
(
 Fixlet Name, Fixlet Site URL, Fixlet ID, 
 Fixlet Action ID, Fixlet Action Script,  Fixlet Relevance, 
 Action Success Criteria, Fixlet Type
 ) 
*/

    name of it
    , url of site of it
    , id of it

    fixlet flag of it

    ) of it
